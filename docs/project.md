# LCSC_android_erp

立创商城的物料器件管理app



包含以下核心功能

## 入库

通过扫描立创物料二维码（例如{on:SO2507139288,pc:C17710,pm:0805W8F4700T5E,qty:100,mc:,cc:1,pdi:166537429,hp:11}）来入库物料

提取出关键信息pc（例如C17710）

去立创商城上搜索该物料
https://so.szlcsc.com/global.html?k=c17710

取得关键信息

```text
0805W8F4700T5E
品牌: UNI-ROYAL(厚声)
封装: 0805
类目: 贴片电阻
编号: C17710
电阻类型: 厚膜电阻
阻值: 470Ω
精度: ±1%
功率: 125mW
描述: 小尺寸、重量轻。降低组装成本,与贴片机匹配。适用于波峰焊和回流焊
```

同时将数量信息保存下来（qty:100）

也支持用户搜索式的入库，用户搜索器件名称，会到呈现立创商城的搜索结果，可供用户选择

入库时用户可以指定保存的仓库位置，一般使用字母加数字表示，例如（A1, C13）

## 搜索

搜索当前物料仓库，支持手动搜索与导入csv文件进行搜索


## 导入导出仓库

可以将本地记录的内容继续导入导出
